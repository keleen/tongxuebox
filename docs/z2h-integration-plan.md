# Z2H 字帖功能整合改造方案

## 一、项目背景

### 1.1 当前项目（童学宝盒）

- **项目名称**：童学宝盒 (tongxuebox_nuxt4)
- **技术栈**：Nuxt 4 + Vue 3 + Tailwind CSS + Nuxt UI
- **核心功能**：
  - 中文汉字字帖生成（基础版）
  - 英文手写练习（四线三格）
  - 简笔画教程
  - 乐谱下载
  - 口算练习

### 1.2 参考网站（Z2H字帖）

- **网站地址**：https://z2h.cn/
- **核心定位**：专业的字帖生成工具
- **功能特点**：功能全面、参数丰富、专业性强

## 四、改造方案

### 4.1 总体策略

1. **保留现有功能**：不破坏现有用户体验
2. **增强核心功能**：重点提升字帖生成的专业性
3. **新增功能模块**：补充缺失的字帖类型
4. **统一交互体验**：参考Z2H的参数设置方式
5. **完善导出功能**：实现PDF导出和打印优化

### 5.2 数据结构设计

#### 5.2.1 字帖配置接口

```typescript
interface CopybookConfig {
  // 基础设置
  text: string; // 输入文本
  gridType: 'tian' | 'mi' | 'none'; // 方格类型

  // 显示设置
  showPinyin: boolean; // 显示拼音
  showStrokeOrder: boolean; // 显示笔顺
  highlightFirst: boolean; // 首字高亮

  // 排版设置
  insertEmptyRows: number; // 插入空行数
  insertEmptyCols: number; // 插入空列数
  gridSize: number; // 方格大小（mm）
  lineSpacing: number; // 行间距（mm）
  charSpacing: number; // 字间距（px，仅段落）
  margin: {
    // 页边距（mm）
    top: number;
    right: number;
    bottom: number;
    left: number;
  };

  // 字体设置
  fontFamily: string; // 字体
  fontWeight: number; // 字体粗细（100-900）
  fontSize: number; // 字体大小（百分比）
  verticalOffset: number; // 上下偏移（百分比）

  // 颜色设置
  traceCount: number; // 描红数量
  traceColor: string; // 描红颜色
  lineColor: string; // 线条颜色
}
```

## 七、注意事项

### 7.1 性能考虑

- PDF生成可能较慢，需要添加加载提示
- 大量汉字渲染需要优化
- 实时预览需要防抖处理

### 7.2 浏览器兼容性

- PDF导出功能需要测试各浏览器
- 打印功能需要CSS优化
- 颜色选择器需要兼容性处理

### 7.3 用户体验

- 参数设置要直观易懂
- 实时预览要准确
- 导出功能要稳定可靠

### 7.4 数据来源

- 拼音数据：使用pinyin-pro库
- 笔顺数据：需要集成汉字笔顺数据库
- 字体：使用Web字体或系统字体
