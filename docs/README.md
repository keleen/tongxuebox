## 🎯 快速概览

### Z2H 网站核心功能

Z2H 是一个专业的字帖生成工具，提供以下功能：

#### 中文字帖类型

- **汉字字帖** (`/hanzi`) - 单字练习，支持笔顺、拼音、描红等
- **词组字帖** (`/cizu`) - 词组练习，支持拼音标注
- **段落字帖** (`/duanluo`) - 段落练习，所见即所得
- **笔画字帖** (`/bihua`) - 基础笔画练习

#### 英文字帖类型

- **字母字帖** (`/alphabet`) - 字母练习，支持大小写成对
- **单词字帖** (`/word`) - 单词练习
- **段落字帖** (`/paragraph`) - 段落练习，连贯性训练

#### 核心参数设置

- 方格类型、方格大小、行间距、页边距
- 字体、字体粗细、字体大小、上下偏移
- 描红数量、描红颜色、线条颜色
- 显示拼音、显示笔顺、首字高亮
- 插入空行、插入空列
- PDF导出、打印功能

### 当前项目状态

**童学宝盒** 目前已有：

- ✅ 中文汉字字帖（基础版）
- ✅ 英文手写练习（基础版）
- ✅ 简笔画教程
- ✅ 乐谱下载
- ✅ 口算练习

### 改造目标

将 Z2H 的专业字帖生成功能整合到童学宝盒中，提升：

1. **功能完整性** - 新增词组、段落、笔画、字母等字帖类型
2. **参数丰富性** - 增加更多可调节参数
3. **专业性** - 实现笔顺显示、描红等专业功能
4. **用户体验** - 优化交互和预览效果

## 💡 技术要点

### 关键依赖

```json
{
  "dependencies": {
    "pinyin-pro": "^3.0.0", // 拼音标注
    "jspdf": "^2.5.1", // PDF生成
    "html2canvas": "^1.4.1" // HTML转图片
  }
}
```

### 技术难点

1. **PDF导出** - 需要处理中文字体嵌入
2. **笔顺显示** - 需要汉字笔顺数据库
3. **实时预览** - 需要精确的尺寸计算和防抖处理
